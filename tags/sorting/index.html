Turbocharging Java With Parallel Array Sorting

29 November, 2016 at 18:22 +0000

Sorting arrays in parallel is a JDK enhancement, which is part of the Java 9 release, which should yield significant performance improvements. JDK enhancement proposal 103 brings a new way of array sorting which uses the Fork/Join parallelism technique to provide sorting of arrays in parallel. This new feature has been added in the Java 8 release. Currently, we have two sorting implementations, one sorts arrays and another sorts collections. These two implementations perform sort in a single thread.
